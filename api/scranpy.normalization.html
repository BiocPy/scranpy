scranpy.normalization.log_norm_counts module#

scranpy.normalization.log_norm_counts.log_norm_counts(x: TatamiNumericPointer | ndarray | spmatrix, block: Sequence | None = None, size_factors: ndarray | None = None, center: bool = True, allow_zeros: bool = False, allow_non_finite: bool = False, num_threads: int = 1, verbose: bool = False) → TatamiNumericPointer[source]#

Compute log normalization.

This function expects the matrix (x) to be features (rows) by cells (columns).

Parameters:

  • x (MatrixTypes) – Inpute matrix.

  • block (Optional[Sequence], optional) – Array containing the block/batch assignment for each cell. Defaults to None.

  • size_factors (Optional[np.ndarray], optional) – size factors for each cell. Defaults to None.

  • center (bool, optional) – center the size factors?. Defaults to True.

  • allow_zeros (bool, optional) – allow zeros?. Defaults to False.

  • allow_non_finite (bool, optional) – all nan or inifnite numbers?. Defaults to False.

  • num_threads (int, optional) – number of threads. Defaults to 1.

  • verbose (bool, optional) – display logs?. Defaults to False.

Raises:

TypeError, ValueError – if arguments don’t meet expectations.

Returns:

log normalized matrix.

Return type:

TatamiNumericPointer
