Run all steps of the scran workflow for single-cell RNA-seq datasets.

  • Remove low-quality cells

  • Normalization and log-transformation

  • Model mean-variance trend across genes

  • PCA on highly variable genes

  • graph-based clustering

  • dimensionality reductions, t-SNE & UMAP

  • Marker detection for each cluster

Parameters:

  • matrix (SingleCellExperiment) A singlecellexperiment.SingleCellExperiment object.

  • features (Union[Sequence[str], str]) Features for the rows of the matrix.

  • block (Union[Sequence, str], optional) Block assignments for each cell. This is used to segregate cells in order to perform comparisons within each block. Defaults to None, indicating all cells are part of the same block.

  • assay (str) Assay matrix to use for analysis. Defaults to counts.

  • options (AnalyzeOptions) Optional analysis parameters.

Raises:

ValueError If SCE does not contain a assay matrix.

Returns:

Results from various steps.

Return type:

AnalyzeResults
